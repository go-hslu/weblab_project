Willkommen

Dieses Projekt wurde im Rahmen des Moduls WEBLAB (Web Programming Lab) an der Hochschule Luzern bearbeitet. Die Projektartefakte sind auf dem GitHub Repository einsehbar.

Mit diesem Technologie-Radar können ähnlich zur Vorlage des Technologie-Radars von Thoughtworks Technologien verwaltet und visualisiert werden. Innerhalb eines Unternehmens wird eine Übersicht auf die in Projekten verwendeten Technologien und Tools verschafft. In einer Visualisierung, ähnlich zu einem Radar, werden Technologien und Tools als Punkte dargestellt. Über die Quadranten beziehungsweise Sektoren werden diese kategorisiert (Frameworks, Tools, Languages, Platforms, Techniques) und über die nähe zur Mitte der Anwendungs-Status (Hold, Assess, Trial, Adopt) definiert.

Es sind Initialdaten und Benutzer erfasst. Mit der E-Mail admin@hslu.com oder cto@hslu.com und dem Passwort 1234 können alle Technologien, auch noch nicht publizierte, eingesehen und editiert werden. Der Benutzer mit der E-Mail user@hslu.com hat keine besonderen Berechtigungen und kann wie ein nicht-authentifizierter Gast nur Technologien ansehen.
